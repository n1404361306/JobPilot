## 四、部署执行步骤
1. 配置生产 `.env`。
2. 执行 `pip install -r requirements.txt`。
3. 执行 `alembic upgrade head`。
4. 执行 `python scripts/init_test_account.py`。
5. 启动 API 服务并检查 `/health`。
6. 启动 Celery Worker 并检查 `health.echo` 演示任务。
7. 构建前端并由 Nginx 托管 `dist`。
